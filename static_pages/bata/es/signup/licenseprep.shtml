Abrir una cuenta de placa

Necesitará la siguiente información para el proceso de suscripción. Por razones de seguridad, tendrá 15 minutos para llevar a cabo el proceso de registro antes de que se agote el tiempo.

Lo que necesitará

Un correo electrónico válido
Tarjeta de crédito
Información de la placa
Es necesario tener una dirección de correo electrónico válida para suscribirse a una cuenta de placa. Se necesita una tarjeta de crédito válida para registrarse en línea. Si quiere abrir una cuenta usando efectivo o un cheque, tendrá que ponerse en contacto con servicio al cliente, para obtener ayuda. Tiene que tener el número de su placa y saberl el estado de cada vehículo que quiera registrar.
